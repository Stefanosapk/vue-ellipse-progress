## How to use
install the library via npm
```
npm i -S vue-ellipse-progress
```
The library is provided as Vuejs plugin. So just initialize it in your ``main.js``
```js
import VueEllipseProgress from '@/plugin';

Vue.use(VueEllipseProgress);
```
